Installation on TARS Robot

  1. Open TARS dashboard at http://your-pi:8000
  2. Go to App Store tab
  3. Enter Space ID: latishab/tars-conversation-app
  4. Click Install from HuggingFace
  5. Configure API keys in .env.local
  6. Click Start
  7. Open dashboard at http://your-pi:7860

Required API Keys

  • DEEPINFRA_API_KEY - For LLM (DeepInfra)
  • SPEECHMATICS_API_KEY or DEEPGRAM_API_KEY - For STT
  • ELEVENLABS_API_KEY (optional) - For premium TTS

Manual Installation

For development or non-TARS deployments:

git clone https://github.com/latishab/tars-conversation-app.git
cd tars-conversation-app
bash install.sh

# Configure
cp env.example .env.local
cp config.ini.example config.ini

# Run
python tars_bot.py  # Robot mode
python bot.py       # Browser mode
