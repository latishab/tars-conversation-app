# TARS Omni

AI brain that connects to Raspberry Pi hardware daemon.

## Pi Access
```
ssh tars-pi  # 100.84.133.74, user: mac, repo: ~/tars-daemon
```

## Run

1. Pi: `ssh tars-pi "cd ~/tars && python tars_daemon.py"`
2. Mac: `python tars_bot.py`
